## Quick View on ProjectLibre

ProjectLibre is an **open source** and cloud company led by Marc O'Brien and Laurent Chretienneau intended as an alternative to commercial software like Microsoft Project.

It is free software, but it is also compatible with any other project management software that can read and write XML-formatted documents.

ProjectLibre desktop's file extention is `pod`.
